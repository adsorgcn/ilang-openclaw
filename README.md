## 安装方式

**腾讯 SkillHub 用户：**

在你的AI里说："帮我在SkillHub安装[技能名]"

**OpenClaw / ClawHub 用户：**

去 [ilang-ai/ilang-openclaw](https://github.com/ilang-ai/ilang-openclaw) 主仓库安装。

**任何AI：**

打开技能目录下的 `prompt.md`，复制全文粘贴到任何AI即可使用。

## CI/CD

Push 到 `main` 自动检测变更的 `skills/*/SKILL.md` 或 `prompt.md`，通过 REST API 发布到腾讯 SkillHub。
